# TASIS Policy Tools

Two-tab tool for policy management:

## Tab 1: Policy Review
- AI-powered compliance checking
- Checks against KCSIE 2025, Working Together 2023
- Validates TASIS template format
- Generates priority action list

## Tab 2: Policy Formatter  
- Structures policies into TASIS template
- **Content is preserved verbatim** (JavaScript parsing, not AI)
- AI only extracts metadata
- Outputs professional HTML with print-to-PDF

## Setup
1. Deploy to Netlify or open index.html locally
2. Enter Anthropic API key
3. Upload policy document

## File Support
- .docx (Word)
- .pdf
- .txt
